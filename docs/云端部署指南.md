# 云端部署指南

## 服务器地址

```
182.92.96.224:18765
```

## 服务器部署

SSH 连接服务器后执行：

```bash
curl -fsSL https://raw.githubusercontent.com/ogurishun2026/NeptuneEqicMcp/main/deploy-cloud.sh | bash
```

## 阿里云安全组

开放端口 18765：
1. ECS 实例 → 安全组 → 配置规则
2. 入方向 → 添加规则
3. 端口：18765，授权对象：0.0.0.0/0

## 本地 Unreal 配置

修改项目根目录 `mcp-config.json`：

```json
{
  "mode": "cloud",
  "local": {
    "host": "localhost",
    "port": 18765
  },
  "cloud": {
    "host": "182.92.96.224",
    "port": 18765
  }
}
```

## 验证连接

1. 重启 Unreal 编辑器
2. Output Log 查看：
   - `Connecting to cloud MCP server at 182.92.96.224:18765`
   - `Connected to cloud MCP server`

## 常用命令

```bash
# 查看服务状态
pm2 status

# 查看日志
pm2 logs neptune-mcp

# 重启服务
pm2 restart neptune-mcp

# 停止服务
pm2 stop neptune-mcp
```

# 版本更新说明

## [1.0.0] - 2026-04-28

### 新增
- 添加 Claude Code skills：`开工` 和 `总结提交`
- MCP 连接测试通过，可通过 Claude 添加 Actor 到 Unreal 场景
- 完整的使用文档和部署说明

### 修改
- 插件名称从 `UnrealMCP` 重命名为 `NeptuneEqicMcp`
- WebSocket 端口从 `8080` 改为 `18765`（避免端口冲突）
- TypeScript 编译配置优化，修复构建错误

### 修复
- 修复 TypeScript 编译时 `pg` 类型定义缺失的问题

### 架构

```
Claude Code → MCP Server (TypeScript) → WebSocket (18765) → Unreal Plugin (C++)
```

### 可用工具

| 工具 | 功能 |
|------|------|
| actor_create | 创建 Actor |
| actor_delete | 删除 Actor |
| actor_find | 查找 Actor |
| actor_list | 列出所有 Actor |
| actor_get_transform | 获取 Transform |
| actor_set_transform | 设置 Transform |
| actor_get_property | 获取属性 |
| actor_set_property | 设置属性 |
| actor_add_component | 添加组件 |
| actor_remove_component | 移除组件 |
| actor_get_components | 获取组件列表 |
